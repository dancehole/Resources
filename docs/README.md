## 一、前言<a id = "title1"></a>

欢迎访问**dancehole**的公共知识库，这里包含常用专业和多方面实用知识，包含Git、Markdown协作、window技巧和杂项类目等。在必要时可以进行快速的文档查阅。同时也作为发布文章、项目文档的备份，做到“工作留痕，学习留迹”。

所有相关开源项目的文档、使用和维护都在此处更新。
